# Housing Assistant

Aplicación para centralizar la búsqueda de alquileres.

## Objetivos

- Buscar anuncios automáticamente.
- Unificar información de múltiples portales.
- Gestionar contactos.
- Priorizar oportunidades.
- Automatizar tareas repetitivas.

## Tecnologías

- Python
- Streamlit
- SQLite
- SQLAlchemy
- Playwright
